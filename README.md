### [Home Lab](https://fboaventura.github.io/)

Showcasing my personal lab setup using Proxmox to simulate production environments for testing and learning. The repository contains detailed documentation, scripts, and configurations is still `private` but will be made `public` as soon as I manage to sanitize and organize everything.

The Hardware used is:

- **Ubiquiti UniFi USG** as the router
- **Ubiquiti UniFi Switch 8** as the main switch
- **Ubiquiti UniFi AP-U6-Mesh** for WiFi
- 3x **Raspberry Pi 4B**
- 1x **Dell Optiplex 7010**
- 1x **Lenovo ThinkCentre M90n**

For the TechStack, I'm using:
- **[Proxmox VE](https://www.proxmox.com/en/proxmox-ve)** for virtualization
- **[Ansible](https://www.ansible.com/)** for automation
- **[Terraform](https://www.terraform.io/)** for IaC
- **[Packer](https://www.packer.io/)** for golden images
- **[Kubernetes](https://kubernetes.io/)** for container orchestration
- **[Helm](https://helm.sh/)** for package management
- **[Traefik](https://traefik.io/)** for reverse proxy and load balancing
- **[Caddy](https://caddyserver.com/)** for web server
- **[Cert-Manager](https://cert-manager.io/)** for SSL certificates
- **[Uptime Kuma](https://github.com/louislam/uptime-kuma)** for monitoring
- **[homepage.dev](https://homepage.dev/)** for the landing page and dashboard
- **[Prometheus](https://prometheus.io/)** for monitoring and data gathering
- **[Grafana](https://grafana.com/)** for dashboards
- **[ELK](https://www.elastic.co/what-is/elk-stack)** for logs
- **[Github Actions](https://github.com/features/actions)** for CI/CD
- **[Docker](https://www.docker.com/)** for containers
- **[Netbox](https://netbox.readthedocs.io/en/stable/)** for IPAM and Inventory
- **[1Password](https://1password.com/)** for secrets management
- **[Keybase](https://keybase.io/)** for secure communications and access control

And I am evaluating:

- **[HashiCorp Vault](https://www.vaultproject.io/)** for secrets management
- **[FluxCD](https://fluxcd.io/)** and **[ArgoCD](https://argo-cd.readthedocs.io/)** for GitOps
- **[Cilium](https://cilium.io/)** for networking and security
